# Art folder

**Note:** Not all of these images are used in the game.

The game currently uses only **`knight.png`** (for the hero and the title screen).

The other images are **alternative art that is not used in the game** — earlier
versions and extra designs kept here for reference:

- `Knight V2.png`, `Knight V3.png`, `Knight pixel art.png` — other knight versions
- `Wizard V2.png`, `wizard.png`, `wizard pixel art.png` — wizard designs (not in the game)
